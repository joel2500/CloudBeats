Connect OneDrive

Enter your Azure App Client ID to connect your OneDrive music library. This is free — no credit card needed.

  1. Go to portal.azure.com → App registrations → New registration
  2. Name it anything, set type to Single Page App (SPA), redirect URI = this page's URL
  3. After creating, copy the Application (client) ID
  4. Go to API Permissions → Add → Microsoft Graph → Delegated → add Files.Read & User.Read
